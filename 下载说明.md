## 🚀 三步启动

### 步骤 1: 安装依赖
```bash
cd financial-trading-system
npm install
```

### 步骤 2: 启动项目
```bash
npm run dev
```

### 步骤 3: 访问系统
- 打开浏览器: http://localhost:3000
- 用户名: **任意** (如: admin, user)
- 密码: **123456**

## ❓ 常见问题

### Q: 如何切换到真实 API？
A: 注释掉 `src/app/api/mock.js` 的导入，配置 `.env` 中的 `VITE_API_BASE_URL`

### Q: 如何添加新页面？
A: 参考 `ARCHITECTURE.md` 中的"添加新页面"章节

### Q: Mock 数据如何工作？
A: 开发环境自动拦截 API 请求，返回模拟数据
